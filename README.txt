Учитељица Бранка — статички сајт
===============================

Ово је статички веб сајт (HTML/CSS/JS) прилагођен да буде спреман за отпремање на хостинг.
Садржи основне странице: Почетна, О мени, Материјали (posts), Галерија, Контакт и Политика приватности.

Фајлови:
- index.html
- about.html
- gallery.html
- contact.html
- privacy.html
- /posts/*.html
- /css/style.css
- /js/main.js
- /assets/*.svg

Како да покренеш локално:
1) Распакуј садржај у фасциклу на свом рачунару.
2) Дупли клик на index.html ће отворити сајт у прегледачу.

Како да га поставиш на интернет:
- GitHub Pages: креирај Git репозиторијум, додај фајлове и пушуј на branch 'gh-pages' или користи репо са pages поставком.
- Било који статички хостинг (Netlify, Vercel, Surge) — откачи садржај и прати процедуру хоста.
- На shared hosting: отпреми фајлове у public_html или www фолдер преко FTP-а.

Напомене:
- Контакт форма користи mailto: (није backend).
- Ако желиш аутоматске обавијести (newsletter) или могућност слања слика преко форме, требаће backend или сервис (Formspree, Netlify Forms).
- Ако желиш да поставим WordPress верзију уместо статичке, могу припремити инсталациони пакет (треба хостинг који подржава PHP/MySQL).
